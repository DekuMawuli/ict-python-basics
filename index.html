Operators & Expressions: Grade Calculations

Arithmetic Operators for Grading

# Student scores
assignment1 = 85
assignment2 = 92
midterm = 78
final_exam = 88

# Calculate final grade
final_grade = (assignment1 * 0.2) + (assignment2 * 0.2) + (midterm * 0.3) + (final_exam * 0.3)
print(f"Final Grade: {final_grade:.1f}%")

# Determine if student passed
passing_grade = 60
has_passed = final_grade >= passing_grade
print(f"Passed: {has_passed}")

Comparison Operators in Education

Evaluating student performance:

Loops: Automating Repetitive Tasks

Processing Class Rosters

# Class roster
students = ["Alice", "Bob", "Charlie", "Diana", "Ethan"]

# Print welcome messages
print("Welcome to Computer Science 101!")
for student in students:
    print(f"Hello, {student}! Welcome to class.")

# Calculate average score
scores = [85, 92, 78, 88, 95]
total = 0
for score in scores:
    total += score
    
average = total / len(scores)
print(f"Class Average: {average:.1f}%")

While Loops for Input Validation

Ensuring valid student input:

# Get valid student score
while True:
    try:
        score = int(input("Enter student score (0-100): "))
        if 0 <= score <= 100:
            break
        print("Score must be between 0 and 100!")
    except ValueError:
        print("Please enter a number!")

Functions: Reusable Teaching Tools

Creating Reusable Grading Functions

# Function to calculate letter grade
def calculate_letter_grade(score):
    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"

# Function to generate student report
def generate_report(name, score):
    letter_grade = calculate_letter_grade(score)
    print(f"Student: {name}")
    print(f"Score: {score}%")
    print(f"Grade: {letter_grade}")
    print("------------------")

# Using the functions
generate_report("Alice", 92)
generate_report("Bob", 78)

Benefits for Teachers

Consistency: Same grading logic for all students

Efficiency: Reuse code across multiple classes

Function Parameters and Return Values

# Function with default parameters
def calculate_final_grade(attendance=100, assignments=0, exams=0):
    # Weighted calculation
    final = (attendance * 0.1) + (assignments * 0.4) + (exams * 0.5)
    return final

# Function with variable arguments
def calculate_average(*scores):
    return sum(scores) / len(scores)

# Using lambda for simple calculations
grade_to_gpa = lambda grade: 4.0 if grade == "A" else 3.0 if grade == "B" else 2.0

Lists and Tuples: Managing Student Data

Lists for Dynamic Collections

# Class roster - mutable collection
students = ["Alice", "Bob", "Charlie", "Diana"]

# Assignment scores
assignment_scores = [85, 92, 78, 88]

# List operations
students.append("Ethan")  # Add new student
students.remove("Bob")   # Remove student
students.sort()          # Sort alphabetically

# List comprehensions for data processing
passing_students = [student for student, score in zip(students, assignment_scores) if score >= 60]

# Calculate statistics
average = sum(assignment_scores) / len(assignment_scores)
highest = max(assignment_scores)
lowest = min(assignment_scores)

print(f"Class Average: {average:.1f}%")
print(f"Highest Score: {highest}%")
print(f"Lowest Score: {lowest}%")

Tuples for Fixed Data

# Course information - immutable collection
course_info = ("Computer Science 101", "CS101", 4, "Fall 2025")

# Student record with fixed fields
student_record = ("Alice Johnson", "S10245", 10, True)

# Accessing tuple elements
course_name = course_info[0]
course_code = course_info[1]
credits = course_info[2]
semester = course_info[3]

# Unpacking tuples
name, student_id, grade, has_passed = student_record

print(f"Course: {course_name} ({course_code})")
print(f"Credits: {credits}, Semester: {semester}")
print(f"Student: {name}, Passed: {has_passed}")

When to Use Lists vs Tuples

Use Lists when:

  • Data needs to change (class roster)
  • Order matters and may change
  • You need to add/remove elements

Use Tuples when:

  • Data is fixed (course info)
  • Using as dictionary keys
  • Returning multiple values from functions

Sets and Dictionaries: Advanced Data Structures

Sets for Unique Collections

# Unique skills in the classroom
skills = {"Python", "JavaScript", "HTML", "CSS", "Python"}

# Set operations
print(f"Unique skills: {skills}")  # Duplicates removed

# Adding and removing skills
skills.add("Database Design")
skills.remove("HTML")

# Finding students with specific skills
python_students = {"Alice", "Bob", "Charlie"}
web_dev_students = {"Bob", "Diana", "Ethan"}

# Set operations
both_skills = python_students & web_dev_students  # Intersection
either_skill = python_students | web_dev_students  # Union
only_python = python_students - web_dev_students  # Difference

print(f"Students with both skills: {both_skills}")
print(f"Students with either skill: {either_skill}")
print(f"Students with only Python: {only_python}")

Dictionaries for Key-Value Data

# Student gradebook
gradebook = {
    "Alice": {"assignments": [85, 90, 88], "exam": 92, "attendance": 28},
    "Bob": {"assignments": [78, 82, 80], "exam": 85, "attendance": 25},
    "Charlie": {"assignments": [92, 95, 90], "exam": 88, "attendance": 30}
}

# Accessing student data
alice_data = gradebook["Alice"]
alice_assignments = alice_data["assignments"]
alice_avg_assignment = sum(alice_assignments) / len(alice_assignments)

# Adding new student
gradebook["Diana"] = {"assignments": [88, 85, 90], "exam": 82, "attendance": 27}

# Calculating final grades for all students
for name, data in gradebook.items():
    assignment_avg = sum(data["assignments"]) / len(data["assignments"])
    final_grade = (assignment_avg * 0.4) + (data["exam"] * 0.5) + (data["attendance"] / 30 * 10)
    print(f"{name}: Final Grade = {final_grade:.1f}%")

Dictionary Methods for Education

# Useful dictionary methods
student_names = list(gradebook.keys())  # Get all student names
all_data = list(gradebook.values())    # Get all student data
items = list(gradebook.items())        # Get key-value pairs

# Checking if student exists
if "Alice" in gradebook:
    print("Alice is in the gradebook")

# Getting data with default value
bob_exam = gradebook.get("Bob", {}).get("exam", "No exam data")

# Updating student data
gradebook.update({"Alice": {"assignments": [88, 92, 90], "exam": 95, "attendance": 29}})

# Removing student
removed_student = gradebook.pop("Bob", None)
