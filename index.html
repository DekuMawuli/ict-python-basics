Python Syntax & Variables

Python emphasizes readability and simplicity

Think of them as sticky labels you slap on jars in the kitchen so you remember what’s inside.

# This is a comment
print("Hello, World!")  # Output text

# Variables - no declaration needed!
name = "Alice"          # String
age = 25               # Integer
height = 5.6           # Float
is_student = True      # Boolean
  • No semicolons required
  • Indentation defines code blocks
  • Variables are dynamically typed
  • Case-sensitive naming

Data Types

  • String = the text you text your friend.

Operators & Expressions

Same buttons you mash on a pocket calculator, but Python does the heavy lifting.

# Arithmetic Operators
result = 10 + 5    # Addition: 15
result = 10 - 3    # Subtraction: 7
result = 4 * 6     # Multiplication: 24
result = 15 / 3    # Division: 5.0
result = 17 // 5   # Floor division: 3
result = 17 % 5    # Modulus: 2
result = 2 ** 3    # Exponentiation: 8

# Comparison Operators
5 == 5    # Equal: True
5 != 3    # Not equal: True
5 > 3     # Greater than: True
5 <= 10   # Less than or equal: True

Control Flow: If Statements

Like deciding what to wear: if it’s cold → jacket, elif it’s warm → t-shirt, else → stay home.

# Basic if statement
age = 18
if age >= 18:
    print("You are an adult")

# If-else statement
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

# Multiple conditions
temperature = 25
if temperature > 30 and temperature < 40:
    print("It's hot!")
elif temperature < 10 or temperature > 40:
    print("Extreme weather!")

Loops: For Loop

Like going through every item on your grocery list and dropping it into the cart one by one.

# Basic for loop
for i in range(5):
    print(f"Count: {i}")  # Prints 0 to 4

# Loop through a list
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(f"I like {fruit}")

# Loop with range parameters
for i in range(1, 11, 2):  # start, stop, step
    print(i)  # Prints 1, 3, 5, 7, 9

# Enumerate for index and value
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

Loops: While Loop

Like repeatedly checking your phone while waiting for the pizza tracker to say “Out for delivery.”

Note: Python doesn't have a do-while loop, but we can simulate it

# Basic while loop
count = 0
while count < 5:
    print(f"Count is: {count}")
    count += 1

# While loop with user input
user_input = ""
while user_input.lower() != "quit":
    user_input = input("Enter 'quit' to exit: ")
    print(f"You entered: {user_input}")

# Simulating do-while behavior
while True:
    number = int(input("Enter a positive number: "))
    if number > 0:
        break  # Exit the loop
    print("Please enter a positive number!")

Functions

Your personal recipe: give it ingredients, it always bakes the same cake.

# Basic function
def greet(name):
    return f"Hello, {name}!"

# Function with default parameters
def introduce(name, age=25):
    return f"Hi, I'm {name} and I'm {age} years old"

# Function with multiple parameters
def calculate_area(length, width):
    area = length * width
    return area

# Using functions
message = greet("Alice")
print(message)  # Output: Hello, Alice!

area = calculate_area(5, 3)
print(f"Area: {area}")  # Output: Area: 15

Lists

Ordered, mutable collections of items

The scribbled sheet you carry to the market—you can add, cross off, or swap items anytime.

# Creating lists
fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, True, 3.14]

# List operations
fruits.append("grape")      # Add item
fruits.insert(1, "mango")   # Insert at index
fruits.remove("banana")     # Remove item
popped = fruits.pop()       # Remove and return last item

# Accessing elements
first_fruit = fruits[0]     # First item
last_fruit = fruits[-1]     # Last item
slice_fruits = fruits[1:3]  # Slice [1:3)

# List comprehension
squares = [x**2 for x in range(1, 6)]  # [1, 4, 9, 16, 25]

Tuples & Sets

  • Tuple = the sealed meal-deal you can’t change once it’s handed to you.
  • Set = a jar of unique buttons—no duplicates allowed.
# Tuples - immutable ordered collections
coordinates = (10, 20)
rgb_color = (255, 128, 0)
person = ("Alice", 25, "Engineer")

# Tuple unpacking
x, y = coordinates
name, age, job = person

# Sets - unordered collections of unique items
unique_numbers = {1, 2, 3, 4, 5}
unique_letters = set("hello")  # {'h', 'e', 'l', 'o'}

# Set operations
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}
union = set1 | set2         # {1, 2, 3, 4, 5, 6}
intersection = set1 & set2  # {3, 4}
difference = set1 - set2    # {1, 2}

Dictionaries

Key-value pairs—like a real word dictionary, but you write the definitions.

# Creating a dictionary
person = {"name": "Alice", "age": 25, "job": "Engineer"}

# Accessing values
print(person["name"])  # Alice
print(person.get("salary", "Not specified"))  # Safe lookup

# Adding/updating
person["city"] = "New York"
person["age"] = 26

# Looping
for key, value in person.items():
    print(f"{key}: {value}")

# Dictionary comprehension
squares = {x: x**2 for x in range(1, 6)}  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Think of it as your phone’s contact list: each name (key) maps to one phone number (value).

Quiz 1: Basic Syntax & Variables

Question 1: Which of the following is a valid Python variable name?

A) 2name
B) my-variable
C) my_variable
D) class
Correct! Variable names in Python must start with a letter or underscore, can contain letters, numbers, and underscores, but cannot be Python keywords.

Question 2: What will this code output?

x = 5
y = "5"
print(x == y)
A) False
B) True
C) Error
D) 5
Correct! x is an integer (5) and y is a string ("5"). They are different data types, so x == y returns False.

Quiz 2: Control Flow

Question 3: What's the output of this code?

score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
print(grade)
A) A
B) B
C) C
D) F
Correct! Since score (85) is >= 80 but < 90, the elif condition for "B" is satisfied first.

Question 4: How many times will this loop run?

for i in range(2, 10, 3):
    print(i)
A) 2 times
B) 3 times
C) 4 times
D) 8 times
Correct! range(2, 10, 3) generates: 2, 5, 8. That's 3 numbers, so the loop runs 3 times.

Quiz 3: Data Structures

Question 5: Which data structure is mutable (can be changed)?

A) List
B) Tuple
C) String
D) Integer
Correct! Lists are mutable - you can change, add, or remove elements. Tuples, strings, and integers are immutable.

Question 6: What's the result of this code?

my_dict = {"name": "Alice", "age": 25}
print(my_dict.get("height", "Unknown"))

B) None
C) Unknown
D) 25
Correct! The get() method returns the default value ("Unknown") when the key ("height") doesn't exist in the dictionary.
